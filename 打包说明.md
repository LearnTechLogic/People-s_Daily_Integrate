# 人民日报爬取工具 - 打包说明

## 打包方法

### 方法一：使用批处理脚本（推荐）
双击运行 `build.bat` 文件即可自动打包。

### 方法二：使用命令行
```bash
pyinstaller --onefile --windowed --name "人民日报爬取工具" gui.py
```

### 方法三：使用 spec 文件
```bash
pyinstaller build.spec
```

## 打包参数说明
- `--onefile`: 打包成单个 exe 文件
- `--windowed`: 不显示控制台窗口
- `--name "文件名"`: 指定输出文件名

## 输出位置
打包完成后，可执行文件位于 `dist` 目录下。

## 注意事项
1. 确保已安装所有依赖：`pip install -r requirements.txt`
2. 确保已安装 PyInstaller：`pip install pyinstaller`
3. 首次打包可能需要几分钟时间
4. 杀毒软件可能会误报，添加信任即可
